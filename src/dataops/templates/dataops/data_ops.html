Upload/Merge

Operation Description
CSV Upload/Merge Upload data contained in a CSV file (typically saved using a spreadsheet or other data manipulation package).
If the table is empty, the data will be simply loaded detecting those columns that have unique values for every row.
If the workflow table has data, the operation will ask you to match a key column in your table with a key column in the file to upload so that the rows can be matched.
Excel Upload/Merge Upload data contained in an Excel file.
Given a file name and the name of a sheet, upload or merge the content of the Excel file.
If the workflow table has data, the operation will ask you to match a key column in your table with a key column in the file to upload so that the rows can be matched.
SQL Upload/Merge Upload data contained in an SQL database providing all the parameters required for the connection and either a query or a table name.
If the table is empty, the data will be simply loaded detecting those columns that have unique values for every row.
If the table already has data, the operation will ask you to match one column in your table with unique values, with another one with unique values from the file that is uploaded so that the columns can be merged with the existing data.
